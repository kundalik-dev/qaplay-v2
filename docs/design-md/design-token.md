# QA Playground — Modern SaaS Design Tokens

Complete token reference extracted from
`Final-UI/05-modern-saas-design-UI/style.css` (the stylesheet that
`modern-qaplay/html-app/index.html` is built on).

- **Dark is the default** (`:root` / `html[data-theme="dark"]`).
- **Light** values apply under `html[data-theme="light"]`.
- The **Tailwind utility** column shows the class generated by
  `tailwind.css` (`@theme inline`). `—` means the token is used inside
  component CSS rather than as a standalone utility.

## Keyframes (carried in component CSS)

`fadeUp`, `fadeIn`, `blink`, `pulse`, `float`, `trustMarquee`,
`testiMarquee`, `ivv-wave`, `ivv-pulse`, `ivv-ring`,
`themeToggleSpin`, `themeIconSwap`, `themeToggleFlash`,
`themeRevealViewport`, `themeRevealExpand`.

These are animation definitions, not tokens — keep them in a global
stylesheet (e.g. the component layer) rather than the Tailwind theme.

## Responsive breakpoints

| Breakpoint               | Behaviour                                                                                            |
| ------------------------ | ---------------------------------------------------------------------------------------------------- |
| `≤ 900px`                | nav links hide / hamburger shows; multi-col grids → 1–2 col; bento → 2 col; section padding tightens |
| `≤ 600px`                | most grids → 1 col; nav name hides; buttons go full-width; section padding tightens further          |
| `prefers-reduced-motion` | animations/transitions reduced to `0.01ms`; marquees & reveals disabled                              |
