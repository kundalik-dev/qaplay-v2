annotations/known-bugs — QA Playground Challenge
Known Bugs & Expected Failures
Annotations Challenge · Section 3 of 7
Intermediate annotations
🎯 Objective
1 Identify the buggy counter (adds 2 instead of 1) and mark the test with test.fixme('BUG-101')
2 Click the flaky action (50% success) and decide whether to use test.fixme() or configure retries
3 Trigger the always-errors button and use test.fail() to confirm the bug still exists
📌 Annotations Used
test.fixme() Skips test, keeps it visible in report as "needs fix"
test.fail() Passes only if the assertion fails — confirms bug exists
🔍 Key Selectors
testid btn-buggy-inc
testid buggy-counter
id #btn-inc / #btn-dec
aria getByLabel('Increase counter')
dyn [data-bug-id="BUG-101"]
testid btn-flaky
testid btn-expected-fail
dyn [data-outcome="always-error"]
💡 Example Test Code
// BUG-101: counter adds 2 not 1
test.fixme('BUG-101', async ({ page }) => {
  await page.click('[data-testid="btn-buggy-inc"]');
  await expect(page.getByTestId('buggy-counter'))
    .toHaveText('1'); // will fail (shows 2)
});

// BUG-202: always-error endpoint
test('always errors', async ({ page }) => {
  test.fail(); // expect failure
  await page.click('[data-testid="btn-expected-fail"]');
  await expect(page.getByTestId('expected-fail-result'))
    .toContainText('Error 500');
});
// Cypress: mark known bugs
it('BUG-101 counter adds 2', () => {
  // Use cy.skip() with cypress-skip plugin
  cy.get('[data-testid="btn-buggy-inc"]').click();
  cy.get('[data-testid="buggy-counter"]')
    .should('have.text', '1'); // fails: shows 2
});

it('always-error button', () => {
  cy.get('[data-outcome="always-error"]').click();
  cy.get('[data-testid="expected-fail-result"]')
    .should('contain.text', 'Error 500');
});
// TestNG — mark known broken tests
@Test(enabled = false,
      description = "BUG-101")
public void counterAddsTwoNotOne() {
  driver.findElement(By.id("btn-inc")).click();
  String val = driver.findElement(
    By.cssSelector("[data-testid='buggy-counter']")
  ).getText();
  Assert.assertEquals(val, "1");
}
